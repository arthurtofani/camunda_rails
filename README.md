# camunda_rails
Easy integration between Rails and Camunda Platform

WIP

## Usage
`rspec spec`

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'camunda_rails'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install camunda_rails
```
